# 产品意图 (Intent)

> 此文件是人类原始意图的不可变记录。禁止 Agent 修改。
> 所有结构化推导请写入 requirements.md。

品牌名称墨简，使用中古风设计的 AI 驱动的简历编辑器，采用中国古风作为设计基准和设计元素
（比如宣纸、墨砚、印章、窗棂、祥云等），采用原子化设计组件，必要时需要网络搜索具体的设计风格。

使用 AI 技术进行驱动，目标是配置多种 API 提供商，初始配置 OpenRouter。

需要考虑用户体验设计，减轻用户认知负担，提供几个 AI 默认优化选项，绑定对应提示词结构。

需要配置多种简历模板，简历使用 Typst 进行渲染。

需要有好看的动画效果。
