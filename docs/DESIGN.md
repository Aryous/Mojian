# 墨简设计系统总纲

> 设计 Agent 的主要参考文件。所有视觉决策从这里出发。

## 设计哲学

墨简的视觉语言来自中国古典文人文化：**宣纸上落墨的克制感，印章的仪式感，窗棂的秩序感**。
AI 功能的现代性不是对抗，而是"持笔者"借助新工具——古风是容器，AI 是内容。

**不是装饰，是语言**：每一个古风元素都有功能对应——宣纸纹理是内容区背景，
印章是确认操作，水墨晕染是加载状态，窗棂是分割与导航。

## 字体体系

- **标题**：衬线体，参考宋体骨架（待选型）
- **正文**：现代黑体，保证可读性优先
- **装饰文字**（如印章、标题装饰）：书法风格字体

## 动画哲学

**水墨物理学**：
- 元素进入：模拟墨水在宣纸扩散，`ease-out`，稍慢（300-500ms）
- 元素消失：墨迹褪去，`ease-in`，较快（150-250ms）
- 悬停：毛笔接触纸面的微压感，`scale(0.98)` + 轻微阴影加深
- 加载：墨水晕染动画（圆形扩散 + 透明度渐变）

禁止：弹性过冲（bounce）、机械线性动画。

## 组件设计语言

- **卡片**：宣纸质感背景 + 细墨边框，圆角极小（2-4px）
- **按钮主操作**：印章风格（朱砂色底，白色文字，方形）
- **分割线**：墨线，带轻微不均匀感（用 SVG 模拟）
- **图标**：毛笔线条风格，非几何形
